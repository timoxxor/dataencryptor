# EVFS — Encrypted Virtual File System

## Что это?
Виртуальный зашифрованный контейнер, в который можно упаковать целую папку с файлами. 
Один `.enc` файл, внутри — все файлы, каждый зашифрован отдельным ключом,
опционально сжаты.

## Ключевые особенности
- **Шифрование AES-256-GCM** — каждый файл со своим nonce и ключом
- **Argon2id + HKDF** — пароль → мастер-ключ → индивидуальные ключи для каждого файла
- **DEFLATE-сжатие** — включается выборочно (пропускает jpg, mp4, zip и т.д.)
- **GUI на egui** — нативный интерфейс с тёмной темой, кастомный тайтлбар, анимации
- **Параллельная обработка** — шифрование через Rayon + поток-писатель
- **Браузер внутри контейнера** — можно открыть `.enc` и просмотреть/извлечь файлы
- **Пул буферов** — переиспользование памяти для больших файлов (с лимитом 64 MB)

## Планы: CAS (Content-Addressable Storage)
Текущая реализация — монолитный контейнер: нельзя удалить или изменить файл,
не пересоздавая всё с нуля. Планируется переход на CAS:
- каждый файл хранится как отдельный блок по хешу содержимого
- дедупликация идентичных файлов
- возможность удалять/добавлять файлы без перепаковки всего контейнера
- сборка мусора для неиспользуемых блоков
